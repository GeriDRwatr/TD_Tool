Привіт. Це TDTool — десктопний PDF-переглядач + редактор (PySide6 + PyMuPDF, Python 3.13).
Користувач — Герід, працює між Windows і Маком, зараз перейшов на Мак.

ПРАВИЛА РОБОТИ В ЦЬОМУ ПРОЄКТІ (не забувай):
- Коміти — українською, дієслово в минулому часі від першої особи ("Переробив", "Виправив", "Додав")
- НЕ додавати "Co-Authored-By: Claude" у коміти
- Після значних змін — оновлювати пам'ять (memory) самостійно без запиту

ПОТОЧНИЙ СТАН ПРОЄКТУ (актуально на момент переходу на Мак):

viewer.py — основний файл, там майже вся логіка переглядача:
  - _FileTabBar: кастомний таббар, setUsesScrollButtons(False), власний _scroll_px offset
    кнопки < > перемикають активну вкладку (не скролять), кнопка + додає PDF
    розташування: [tabs(flex)] [+] [<] [>] — кнопки справа від вкладок
  - _ThumbnailPanel: ліва панель мініатюр, синхронізується з поворотом сторінки
    (сигнал page_rotation_changed → update_rotation з rotation-aware scale)
  - ScreenViewer: переглядач, тулбар, статус-бар
  - PdfViewerTabs: контейнер вкладок + shared sidebar

ЩО МОЖЕ ПОВОДИТИСЬ ІНАКШЕ НА МАК:
  - app/platform.py — Windows-only код, на Маку автоматично no-op (не падає)
  - Кольори тайтлбару (set_title_bar_color) — тільки Windows, на Маку ігнорується
  - DPI: на Retina дисплеї devicePixelRatio=2 — рендеринг вже враховує це через _device_pixel_ratio()
  - Шрифти: sf_font() намагається знайти SF Pro, на Маку він буде доступний нативно

ЯК ПОЧАТИ РОБОТУ:
  cd TD_Tool && source venv/bin/activate && python main.py

=== КІНЕЦЬ ІНСТРУКЦІЇ ДЛЯ CLAUDE ===

Привіт. Продовжуємо роботу над проєктом TDTool — PDF-переглядач + редактор на PySide6 + PyMuPDF (Python 3.13).

Репо: https://github.com/GeriDRwatr/TD_Tool.git

--- СИНХРОНІЗАЦІЯ МІЖ ПРИСТРОЯМИ ---

На Windows (перед тим як перейти на Мак):
  git add -A
  git commit -m "..."
  git push

На Маку (щоб отримати свіжий стан):
  git pull
  (якщо після pull помилка "diverged history" або "rejected" — значить на Windows була перезапис
  історії через force push; тоді виконай:)
  git fetch origin && git reset --hard origin/main


--- РОЗГОРТАННЯ НА WINDOWS ---

git clone https://github.com/GeriDRwatr/TD_Tool.git
cd TD_Tool
python -m venv venv
venv\Scripts\activate
pip install PySide6 PyMuPDF python-docx pywin32-ctypes
python main.py


--- РОЗГОРТАННЯ НА МАК (перший раз) ---

git clone https://github.com/GeriDRwatr/TD_Tool.git
cd TD_Tool
python3 -m venv venv
source venv/bin/activate
pip install PySide6 PyMuPDF python-docx
python main.py

  Примітки:
  - pywin32-ctypes НЕ встановлювати — Windows-only, на маку не потрібен
  - Якщо python3 не знайдено: brew install python
  - Якщо PySide6 не встановлюється: перевір що Xcode Command Line Tools є (xcode-select --install)


--- НАСТУПНІ СЕСІЇ НА МАК (якщо venv вже є) ---

cd TD_Tool
source venv/bin/activate
git pull
python main.py

--- ЩО РЕАЛІЗОВАНО В ПЕРЕГЛЯДАЧІ (viewer.py) ---

- Ліва панель мініатюр + зміст (TOC) — перемикається правою кнопкою на іконці sidebar
- Режими сторінок: неперервне / окрема сторінка / дві сторінки
- Зум: Ctrl+колесо / pinch-жест тачпаду / кнопки тулбару
- HiDPI/Retina рендеринг (devicePixelRatio × fitz.Matrix)
- Виділення тексту: 2 кліки — слово, 3 — рядок, 4 — абзац
- Пошук по документу: case-insensitive для кирилиці
- Тулбар: info, save (експорт з поворотами), zoom по центру, share/markup/rotate/T
- Поворот поточної сторінки; окрема кнопка збереження зі збереженням поворотів
- Drag-and-drop PDF прямо на переглядач


--- ВАЖЛИВІ НЮАНСИ ---

- app/platform.py — Windows-specific (pywin32-ctypes). На macOS/Linux no-op, не падає.
- app/screens/window.py — НЕ main.py (перейменовано щоб уникнути конфлікту з кореневим main.py)
- Іконки: app/ui/icons/ — Python-пакет. draw(painter, rect, name, color) сам обирає SVG або
  vector. Окремо імпортувати svg_icons не потрібно.
- Lucide/ папка (якщо є на диску) — локальне джерело іконок, в .gitignore, в репо не входить.
- window_state.json (розмір вікна) і app/theme.json (тема) — в .gitignore, runtime-файли.
